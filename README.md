# 中文版

**在定制的开放世界中重新定义你的大逃杀战场**

核心特色：
- 随机生成的开放世界：开箱即用的原版地形生成、不同群系有独特的掩体数量与物资点
- 高度自定义的区域：自由设置平面及立体区域形状、并可添加多种特殊效果
- 自定义物资生成：可组合任意物品和实体刷新，支持原版箱子刷新

该整合包是在完整自定义大逃杀（整合包）的基础上添加了原版地形生成，无需担心没有合适的游戏地图
